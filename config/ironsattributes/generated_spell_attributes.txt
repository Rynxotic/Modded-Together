===========================================
SPELL ATTRIBUTES
===========================================
These control whether a player can cast individual spells.

When spellLevelGating is DISABLED (default):
  Value: 0 = locked, 1 = unlocked

When spellLevelGating is ENABLED:
  Value = max spell level the player can cast
  0 = locked, 1 = up to level 1, 2 = up to level 2, etc.
