# Patch Notes — 1.2.6 Update

> Bug fixes, new content, balance changes & more

## New Content

- Added **Wireless Crafting Terminal**
- Added **Refined Types**
  - Allows importing/exporting Energy (FE) and Source (Ars) wirelessly
- Added **Create Filters Anywhere**
  - Create Filters now work in Sophisticated Storage/Backpacks and Refined Storage
- Added new **Warden variants**
- Added **Drygmy**
  - Containment ritual power is low, allowing only weaker mobs to be trapped
  - Full list at:
    `moddedtogether/kubejs/server_scripts/jar_blacklist.js`
- Added new crafting sounds

---

## Updates

- **Pure Darkness** (Forlorn Hollows Cave) is now used for Shadow Spells
  - Nightmare Fuel removed
- **Interdimensional Wireless Transmitter** recipe changed
- **Zenalite Ingot** recipe revisited
- **Roll (dash)** is now possible mid-air
- Various fixes from mod updates

## Balance Changes

- Added some **Alex Caves** blocks to the Spectral Hammer whitelist

## Removals

- Removed **Ghost Step Perk** from Affinity
  - Conflicted with Tombstone
- **Spirit Orbs / Goddess Statue** disabled in caves
- **Spark** removed
  - Left as optional install
- **Cosmetic Armors** (Miku, etc.) removed
  - Hazenn removed them alongside critical patches
  - Stability had to come first
- Removed **Ghost Sword** and **Phantasmal Ingot**
- **Blur** re-removed
