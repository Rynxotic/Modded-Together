# Apotheosis & Progression

* World tiers were removed and the entire system was reworked:
  * Common / Uncommon **removed**
  * **Rare**, **Epic**, **Mythic** only
* **Rare** = natural drops
* **Epic / Mythic** = Gateways, chests, Trial Chambers (all dimensions)
* Bookshelves are disabled intentionally (balance).

## Enchants

* **Scavenger**, **Knowledge of the Ages**, etc.:
  * Not obtainable via enchanting.
  * Only found on existing items.
