# Ars Storage

## Getting Started

To get started you will need a Storage Secturn, a Bookwyrm Charm and a Dominion Wand.

* Storage Lectern
<BlockImage id='ars_nouveau:storage_lectern'/>
* Bookwyrm Charm
    <ItemImage id='ars_nouveau:bookwyrm_charm'/>
* Dominion Wand
    <ItemImage id='ars_nouveau:dominion_wand'/>

## Setting up Storage
- Place down the Storage Lecturn
- Right click Storage Lecturn with Bookwyrm Charm
- Right click Chest or Storage Controller with Dominion Wand
- Right click Storage Lecturn with Dominion Wand

And now you have a fully working storage system, if you require more chests add more Bookwyrm Charms
