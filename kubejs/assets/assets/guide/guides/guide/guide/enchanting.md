# Enchanting
You can extract enchants using:

* **Blaze Forger**

  <RecipeFor id="create_enchantment_industry:blaze_forger"/>

* **Enchantment Templates** 

  Enchanting Template
  <ItemImage id="create_enchantment_industry:enchanting_template" />
  Super Enchanting Template
  <ItemImage id="create_enchantment_industry:super_enchanting_template" />
## Process

1. Insert item
2. Insert template
3. Extract enchant → stored on template

Templates can be reused or applied to books.
