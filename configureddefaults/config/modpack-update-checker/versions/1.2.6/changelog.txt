--------------------------------------------------
New Content
--------------------------------------------------

- Added Wireless Crafting Terminal.

- Added Refined Types integration.
  Allows wireless importing/exporting of:
  - Forge Energy (FE)
  - Ars Nouveau Source

- Added Create Filters Anywhere support.
  Create filters now function with:
  - Sophisticated Storage
  - Sophisticated Backpacks
  - Refined Storage

- Added new Warden variants.

- Added Drygmy containment adjustments.
  Containment Ritual power is intentionally reduced and can now only trap weaker entities.

  moddedtogether/kubejs/server_scripts/jar_blacklist.js

- Added new crafting sounds.
